How do I install the repocket app?

 

The latest version of the Android app is always available for download here: https://repocket.co/download/
You will be downloading and installing an APK file; later, Repocket will become available on the Google Play store.
How do I update the Android app?
You will get a notification on your android phone if there is a version which you can update into. Alternatively, you can download the latest version of the APK from this link https://repocket.co/download/
The new version of the app will replace the existing version automatically.
I am experiencing an issue with the app; how do I fix this?
Firstly, please keep in mind that we are in Beta.
If you are experiencing an issue, make sure you have the latest version of the app installed, as this will fix most issues.
If you have the latest version of the app installed and are still experiencing issues, contact us, and we'll provide you with one-on-one support to fix any issues.
How frequently is reporting updated?
All reporting is updated daily; this includes referrals, bandwidth used and earnings.
We will update the reporting frequency in a future release.
Can I use the same account on multiple devices?
Yes, you can use the same account on as many devices and connections as you have. Log in to the same account for each device you want to use. Note: 1 IP = max of 2 devices only.
Why can't I see multiple devices signed in?
There are instances when some devices aren't showing on the devices tab. You'll still be able to share the internet and earn but it might not show on the devices tab. This will be updated soon.
